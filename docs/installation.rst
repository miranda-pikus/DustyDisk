Installation
=============

This package can be installed as below, irrespective of the operating system.
.. code-block:: bash
	
	$ pip install DustyDisk

Make sure to install the requirements such as ``numpy`` and ``matplotlib``. 

For Developers
++++++++++++++

``DustyDisk`` is in a rudimentary stage with minimal features. The following method for 
installing ``DustyDisk`` will allow you to modify it and make user friendly changes to it. 

After cloning the Git repository, run the following commands.

.. code-block:: bash
	
	$ pip install -r requirements.txt
	$ pip install -e . --upgrade
