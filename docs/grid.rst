Functions
==============

#. 'Constants.py' has all the astrophysical constants and right now everything is in cgs.
#. 'Functions.py' is the major file which would help the user to initiate the system (Initialize_System()), find drift veloctiy (vdrift())
and dust density (dust_denstiy()).
#. 'Plotting.py'  has different functions to generate the plots of physical quantities such as pressure,density and drift 
velocity of the dust.
